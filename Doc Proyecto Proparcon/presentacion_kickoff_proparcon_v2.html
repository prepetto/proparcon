  • Análisis de seguridad en licencias Business Basic.

  • Automatización de procesos con PA (Power Automate).

  • Prototipado rápido sobre Excel/Listas.

  • Configuración de entornos dev y pro.
  • Sustitución de Docker local para el desarrollo de P3.

P3: Stack Web Final

  • API con FastAPI y Frontend con React + Tailwind.
  • Ciclo de pruebas directamente en Cloud.
