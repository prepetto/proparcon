Proyectos G1 Prioridad Alta

  • P1: Auditoría Tenant Office 365 y Seguridad MFA.
  • P2: MVP Power App Gestión Patrimonial.
  • P4: Escenario Cloud (Supabase/Render) - Req. previo P3.
  • P3: Stack Web Final (Postgres + FastAPI + React).

P1: Auditoría O365

  • Análisis de seguridad en licencias Business Basic.
  • Endurecimiento de identidad: MFA obligatorio.
  • Automatización procesos con PA (Power Automate).
  • Auditoría de registros DNS (SPF, DKIM, DMARC) para entregabilidad.
  • Revisión de permisos de compartición externa en SharePoint.

P2: MVP Power App

  • Prototipado rápido sobre capa de datos Excel/Listas.
  • Formularios de alta y gestión de inmuebles/habitaciones.
  • Lógica de negocio: Flujos de aprobación en Power Automate.
  • Validación de procesos reales con el responsable funcional (Juan).

P4: Escenario Cloud

  • Setup de Supabase como persistencia remota (Postgres).
  • Despliegue de servicios en Render para el API y Web.
  • Configuración de entornos segregados (dev / pro).
  • Pipeline de CI/CD automático desde ramas de GitHub.

P3: Stack Web Final

  • Migración del modelo a Esquema ER V2.0 en Postgres.
  • API robusta con FastAPI y validaciones Pydantic.
  • Interfaz moderna y responsive con React + Tailwind.
  • Fase de Seed y QA integral (DB → API → Frontend).

Proyectos G2 Prioridad Baja

  • P5: MVP Aplicación Inventario (Lovable).
  • P6: Intranet SharePoint Proparcon.
  • P7: Scraping Idealista (Refactorización local).
  • P8: Infraestructura Proxmox, VPN y n8n Local.

P5: MVP Inventario

  • Uso de herramientas de generación rápida con IA (Lovable).
  • Gestión visual de activos, muebles y enseres por estancia.
  • Captura de fotos para estados de check-in / check-out.
  • Exportación de datos para integración futura.

P6: Intranet SharePoint

  • Portal central de comunicación interna para Proparcon.
  • Bibliotecas de manuales, procesos y documentación técnica.
  • Dashboard de accesos rápidos a las aplicaciones del Grupo 1.
  • Uso de Sitios de Comunicación para máxima usabilidad.

P7: Scraping Idealista

  • Refactorización y limpieza del script Python actual.
  • Migración a contenedor Docker en servidor local.
  • Optimización de tiempos para evitar detección de bots (Bypass).
  • Documentación técnica para ejecución bajo demanda.

P8: Proxmox & Redes

  • Setup de Hypervisor Proxmox VE sobre hardware local.
  • Túneles seguros con VPN Wireguard para acceso remoto.
  • Automatizaciones internas con instancia local de n8n.
  • Servicios críticos: DNS interno, Proxy inverso y Firewall.
